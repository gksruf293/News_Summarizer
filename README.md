# 📰 AI News English Study
> **AI가 매일 배달하는 맞춤형 뉴스 요약 서비스** > 사용자의 수준에 맞춘 3단계 요약으로 영어 실력을 스마트하게 키워보세요!

[![Web Site](https://img.shields.io/badge/Visit-My%20App-blue?style=for-the-badge&logo=googlechrome)](https://gksruf293.github.io/news_summarizer/)

## 🚀 서비스 소개
단순한 뉴스 읽기를 넘어, **GPT-4o-mini**를 활용한 지능형 요약과 **시맨틱 검색** 기능을 제공합니다. 
자신의 영어 수준에 맞는 뉴스를 골라 읽으며 가장 효율적으로 학습하세요.

### ✨ 주요 기능
- **📊 3단계 레벨별 학습**: 초등(3줄), 중등(5줄), 고등(7줄) 수준의 맞춤형 요약 제공
- **🔍 AI 시맨틱 검색**: 단순 키워드가 아닌 의미 기반의 뉴스 검색 가능
- **📅 날짜별 뉴스 아카이빙**: 달력을 통해 과거 뉴스를 언제든 복습
- **🇰🇷 실시간 번역**: 영문장 클릭 한 번으로 한국어 해석 확인

---

## 🛠️ Tech Stack
- **Frontend**: Vanilla JS, HTML5, CSS3
- **Backend (Automation)**: Python, GitHub Actions
- **AI Models**: 
  - `GPT-4o-mini` (Text Summarization)
  - `Xenova/all-MiniLM-L6-v2` (Browser-side Semantic Search)
